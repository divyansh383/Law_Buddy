# Legal Buddy

Legal Buddy is an intelligent assistant designed to accurately understand and respond to legal queries. Leveraging Generative AI (GenAI) and Retrieval-Augmented Generation (RAG), the chatbot is trained on a comprehensive dataset of Supreme Court case studies and other legal documents.

## Key Features

- **NLP Integration**: Process and understand user queries in natural language.
- **RAG Implementation**: Retrieve relevant case information and generate appropriate responses.
- **Case Study Training**: Utilize a dataset of Supreme Court case studies and other legal texts for training.
- **User Interface**: Develop an intuitive and user-friendly interface for interaction.
- **Continuous Learning**: Regularly update the chatbot with new cases and legal information to enhance its performance.

## Installation and Setup
   ```bash
   git clone https://github.com/divyansh383/Law_Buddy.git
   cd Law_Buddy
   python -m venv myenv
   myenv\Scripts\activate
   pip install -r requirements.txt
   streamlit run main.py
  ```
